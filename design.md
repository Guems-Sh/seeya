# Design System SeeYa

## Couleurs
- Background: #000000 (noir pur)
- Card/Surface: #1A1A1A (gris très foncé)
- Primary Accent: #CCFF00 (lime neon)
- Text: #FFFFFF (blanc)
- Text Secondary: #999999 (gris)
- Border: #333333 (gris moyen)

## Typography
- Heading: Bold, sans-serif (Arial Black)
- Body: Regular, sans-serif
- Button: Bold, uppercase

## Spacing
- Padding standard: 16px
- Gap elements: 12px
- Border width: 2px (thick borders)

## Components
- Button: lime background, black text, bold, thick border
- Card: white background, black border 2px, padding 16px
- Input: white background, black border 2px, padding 8px
- Bottom Nav: 5 icons (Home, Explore, Drop, Me)
- Tabs: white/black toggle style (PENDING, CONFIRMED, PAST)

## Breakpoints
- Mobile: 375px (primary)
- Tablet: 768px
- Desktop: 1024px

## Key Pages
- Drop a Pin: Create event form (planned or right now)
- Inbox: Pending/Confirmed/Past tabs
- Welcome: Onboarding (avatar, name, handle, bio, vibes)
- Map: Interactive Paris map with events + filters
- Event Detail: Event card + Join button
